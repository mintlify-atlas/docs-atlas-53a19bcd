title: "Catalyst: Declarative C/C++ Build System"
description: "Catalyst brings Cargo-like simplicity to C and C++ projects. Replace CMake and Makefiles with clean YAML configuration and reproducible builds."
---

Catalyst is a declarative meta build system for C and C++. Define your project in a `catalyst.yaml` file and let Catalyst handle compilation, dependency management, and build orchestration.
