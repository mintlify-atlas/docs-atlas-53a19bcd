## Global options

These flags are accepted by every Catalyst command:

| Flag | Description |
|------|-------------|
| `-v, --version` | Print version information and exit. |
| `-V, --verbose` | Enable verbose logging (debug output). |
| `-h, --help` | Print a help message for the command and exit. |
